Website Creation

Reference for how to create a website such as this one using GitHub pages and a template


GitHub pages provides a free option for website hosting, and obviously has integrated version control. By using a template you can quickly and fairly easily get a decent (debatable) looking website up and running. I'll run through the basic steps needed to generate a site from scratch as a reference for myself an maybe others. There's lots more to learn with html, CSS etc., but I'll save that for future me to write about.

Step 1
Log into or set up your GitHub account.

Step 2
Create a new repository. Name it username.github.io, where username is your GitHub username.
E.g. mine would be robin-clancy.github.io.
N.B. I had to make this repository public for the website to be deployed.

Step 3
CHead over to html5up.net. Pick a theme you like and download it. I used the "Landed" theme.
You'll be able to customise the images and many aspects of the appearance, so focus more on the general layout of the theme.

Step 4
Unzip the downloaded theme. Upload these files to the repository you created. There are two ways to do this depending on how comfortable you are with GitHub.
The fancy way, and one which you'll eventually want to learn involves cloning your repository so you have a copy of it on your desktop, moving the files into there, then using git add -A, git commit, then git push. If that sounds like nonsense to you, then going to https://github.com/username/username.github.io (again, swap your username in), clicking add file, upload files allows you to use the GitHub website user interface, which is pretty simple.

Step 5
You should now have a working website, albeit a prety generic one. Visit username.github.io. (once again, change username to your username) to check it out.
If it's not there yet then sometimes it helps to wait a minute or two for it to deploy and try again.
The last deployment time can be seen under github.com/username/username.github.io/settings/pages.

Step 6
You've done the hard part. Now you just have to learn HTML, CSS etc. etc. so you can tweak the website to be a bit more personal. I'll leave that up to you, but here are a few tips:
1. Using right click ➞ inspect will allow you to see the code behind various features in your own or other people's websites, which can reveal how to tweak your own features or borrow someone else's features and code.
2. The .html files can be opened in VS Code for easy editing.
3. The .html files can be opened in a browset to see the results of any changes you make.
4. Details of the CSS code that defines the underlying style of the template can be found in: assets ➞ css ➞ main.css.
5. I used Runway's text-to-image product to make a few starter images that hopefully aren't breaking any copywrite laws.
6. To update the website on GitHub you can use: git add -A, git commit, then git push. Then wait a minute or so for it to deploy.
